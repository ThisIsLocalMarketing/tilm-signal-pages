# Signal Page Scope

## Basic Signal Page

A simple mobile action page for one business or offer:

- business name/logo,
- one clear offer,
- call button,
- directions button,
- website/menu/booking/review/social link buttons,
- offer terms/fine print,
- proof approval before public use.

## Custom Signal Page

A more polished page for stronger placements:

- everything in Basic,
- stronger layout/copy,
- multiple sections if needed,
- custom visual treatment,
- trust path notes if appropriate,
- QR Physical Display tie-in if included.

## Claim boundary

Use scan/action language only when actually trackable. Do not claim guaranteed sales, revenue, ROI, calls, rankings, or redemptions.
