Current pilot

This Is Sutton founding card.

The first Sutton run is planned as a premium full-color 9x12 mailed card built to be seen, kept, and scanned, with 5,000 mailed cards promoting 17 local small businesses: 1 Anchor, 3 Featured, and 13 Standard spots. The card gives each advertiser shared local visibility, while the This Is Sutton Signal Page — a simple scan/click page for the offer — gives interested customers a clear place to learn and act.

Mailing planSutton residents are prioritized first. If any cards remain, the route expands to nearby higher-income homes. The 9x12 size gives each offer more room to stand out.
Best fitBuilt for local service, food, wellness, retail, home, and community-facing businesses that want one clear offer in front of nearby residents.
Print-ready rulePrint readiness is reached when cleared, proof-approved revenue covers the verified all-in shipped-card cost and final artwork is ready. It is not tied to a fixed sold-slot count.
Standard · $444Designed card spot with one clear offer and an action path within the This Is Sutton Signal Page.
Featured · $555Stronger visual weight than Standard, with the option to include a QR code inside the ad when that helps the offer.
Anchor · $777Premium top placement for the first issue, built around one strong local offer and a clear action path using a custom QR code.
Proof-first processCategory fit and offer direction are reviewed first. A proof is drafted for approval, and payment is requested only after the advertiser approves that proof.
Category fitEach main category is limited to one business, so direct competitors are not placed together on the card. Similar-but-different services are reviewed clearly before any proof or payment step.
