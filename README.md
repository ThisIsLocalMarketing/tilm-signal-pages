# TILM Signal Pages

Public-safe Signal Pages and static launch pages for **TILM / This Is Local Marketing**.

## Current purpose

This repository starts with the combined **TILM + This Is Sutton** launch Signal Page and reusable Signal Page templates.

## Public-safety rule

This repository is intended to be public for GitHub Pages hosting. Do **not** commit:

- private advertiser/prospect lists,
- customer private contact data,
- Stripe/payment information,
- API keys/secrets,
- personal/private phone numbers unless intentionally public,
- private vendor quotes,
- unapproved advertiser assets.

Use this repo only for public-safe pages, templates, approved assets, and non-sensitive code.

## Forgotten ethics guardrail

All pages must avoid guaranteed sales, guaranteed ROI, guaranteed rankings, fake scarcity, manipulative urgency, broad unsupported exclusivity promises, review incentives, and private/sensitive data exposure.

Allowed emphasis: local visibility, clearer offers, useful action paths, scan/call/book/review/find paths, and conservative tracking language.
